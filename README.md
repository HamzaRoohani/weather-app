Weather App

A simple Weather application built with Flutter and GetX that fetches and displays current weather data from the OpenWeatherMap API. This project demonstrates REST API integration, state management, and responsive UI design.

Features
- Search for any city to get current weather.
- Display temperature, humidity, description, and weather icon.
- Loading and error handling states.
- Clean and responsive UI with navigation between screens.

Technologies Used
- Flutter
- GetX (State Management)
- Dart
- HTTP package for API calls
- OpenWeatherMap API
